# R-shiny
Collections of R shiny apps made by myself

**1. KM.R: Kaplan-Meier Survival Plotter** 
- Plot K-M curves using uploaded data, after selecting event time and censor status. Support interaction with stratification for multiple combinations of groups and data filtering.
- shinyapps.io link: https://yuezhan.shinyapps.io/KMcurve/

**2. Volcano.R: Volcano plotter for 2 or 3+ groups**
- Draw Volcano plot of genes by uploading a wide format data, using T-test for 2 groups or ANOVA followed by Tukey's HSD post-hoc test for over 3 groups.
- shinyapps.io link: https://yuezhan.shinyapps.io/Volcano/
